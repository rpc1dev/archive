-Affrey DD-4008E (8X/32X) 41
(Known firmware : 10 / 12 / 40)
If you have the RPC-2 firmware 40, you can make the drive to RPC-1 (region free). The trick is to put a jumper on the middle set of pins to the left of the digital audio out (on the far left of the back of the drive).
If you update or already have the firmware 41, which you have to anyway cause of it's reading problem with firmware 40... then set the jumper to exactly the same place which described for the old firmware 40. Then eachtime you boot your pc, hold the two buttons on the frontside of the dvd-panel pressed, then the drive became absolutely codefree (RPC-1). NOTICE: you have to do this everytime you boot up! otherwise it will just keep the old RPC-2 settings.

-Matshita (Panasonic) SR-8583 (5X/32X) C53H, C54A, C61C, E61C, C61f / 0W01, 0Y01

0W01 allow you to upgrade your drive from 5X to 6X DVD reading. Note : This firmware do not work with DVD-RAM. If you want to use them use the C61f firmware.
0Y01 is identical to 0W01 but is U/DMA. Here is a page with comparative results between C61f (5X) firmware and 0W01 (6X).
E61C is an Ultra-DMA version but has problem with some DVD software.
For mac users : C54A, C61f / 0W01, 0Y01 (Mac flash utility included)
These firmware could also be used with the SR-8583 drive include in Mac G3 to make the drive region free.
Once done, the reading of the DVD zone 1 and 2 or other does not pose any more problem because the Apple software zone protection is based on the DVD firmware
Tested on MacOS 8.6 with DVD player 1.2 OS integrated (Thanks to Alain).
In some case you need a hack to reset the software region code. Available at DVDutils (see link page).
To be flashed correctly the drive must be connected on the secondary IDE Port with the drive set as Master.
