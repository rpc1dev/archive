firmware

If your DVD drive has a newer firmware send me a mail.
Bad flashing could destroy your drive definitively.
Read carrefully the installation notes of your firmware upgrade.
At your own risk!!!!

Don't forget you also need a region free DVD software!!! See the link page to find them.
If you have questions please read the DVDutils FAQ first.

RPC-1 = Region free drive
RPC-2 = Region locked drive
To know if your drive is region protected use driveinfo in the utility page. (Please report me all drive/firmware not present in the list).

-Matshita (Panasonic) SR-8583 (5X/32X) C53H, C54A, C61C, E61C, C61f / 0W01, 0Y01
(Know firmware : C53H / E53H / C54A / D54A (G3 drive zone locked)/ C61C / E61C / 061C / C61f / 0W01 / 0Y01)
0W01 allow you to upgrade your drive from 5X to 6X DVD reading. Note : This firmware do not work with DVD-RAM. If you want to use them use the C61f firmware.
0Y01 is identical to 0W01 but is U/DMA. Here is a page with comparative results between C61f (5X) firmware and 0W01 (6X).
E61C is an Ultra-DMA version but has problem with some DVD software.
For mac users :
These firmware could also be used with the SR-8583 drive include in Mac G3 to make the drive region free. Since there is no flasher for MAC, flashing must be done on a PC.
Once done, the reading of the DVD zone 1 and 2 or other does not pose any more problem because the Apple software zone protection is based on the DVD firmware
Tested on MacOS 8.6 with DVD player 1.2 OS integrated (Thanks to Alain)
To be flashed correctly the drive must be connected on the secondary IDE Port with the drive set as Master.
