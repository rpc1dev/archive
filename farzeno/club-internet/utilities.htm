Utilities

For PC

- CDVDInfo 1.35 To check if your DVD drive is region protected or not!! Highly recommanded!

- Driveinfo v2.1 is an utility (written by Erwin Van Den Berg) also gives the firmware version you have and show you if your DVD drive is region protected.

- Nero info tool 1.00 It's a utility which analyses and displays the most important informations about a drive, disc, configuration and software.

- Mhsysstat v1.03 is a very useful utility who could show you the firmware date and version for all your drives and more other stuff.

- Toshiba region change utility (5 times only) Works with all RPC-2 drives.

- Ricoh region change utility (5 times only) Very similar with the Toshiba utility. Should also works with all RPC-2 drives.

- Hitachi region change utility (5 time only) Only works on Hitachi drives. You have a description of the utility here.

For Unix/Linux

Rpcmgr 1.1 : Equivalent to Driveinfo on PC. Can also set LG and Hitachi drives to RPC-1 mode. Written by Dag Lem

For Macintosh users

DVDInfo 2.0.3 : (Equivalent to Driveinfo on PC) Let you know the brand, model, and firmware version of your DVD drive, and if the drive is RPC-1 or RPC-2 (and in this case, the number of changes allowed).
Region Switch 1.0.9 : Let you see and change the region of the Macintosh software player (only if the drive is RPC-1). No need to reboot!
Do not works with MAC OS X !!

RegionX 1.0.4 by XVI for MAC OS X. Region X allows you to control the region settings of your OS X DVD Player with region free drives. It will run on OS X only, so it does not replace the old Region version intended for Mac OS Classic.

XVI stuffs

How to patch a DVD firmware for region free v1.3 (PDF)

Disassembler source code (contain Hitachi H8, Matsushita 102000, Mitsubishi 7902, Hitachi superH, Z180 and 8051 disassembler)

Region Display (PAL or NTSC version) a very small DVD movie that will display your player region settings, as they may be queried by some movies, e.g. RCE protected movies. Suitable for checking topset DVD players.
