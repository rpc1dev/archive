With Sandra i found that is FCPGA (otherwise A360). Now which is the difference between them?

So i decided to try someone else's firmware. With no info about how to do that i read bit of RPC1 forum. Found few firmware's from other people, bit of info how to do that and started to try.
Firstly i get mpathc.exe from RPC1 page (v1.12b) and with manual option tried "drvloadc.exe /?"

In my computer sometimes happens that when i'm working in pure DOS sometimes in freezes in some of DOS versions. Cant remember in which one. Downloaded DRVLOADC2.EXE file tried with it and there was same results (both drvloadc and drvloadc2 are v2.54).
So i tried to patch it with TTX150_710.zip can't remeber where i got it but it worked with /SM option from RamDrive. But doing dvrloadc.exe (from mpatch.exe v1.12b) /M TTX150.710 (both in Ram Drive) get this error:
