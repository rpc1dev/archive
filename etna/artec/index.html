Flashing your drive! (Works only with real DOS-MODE)



Create a Bootdisk and unzip Ultima Flasher and a firmware to the root directory of the bootdisk.
Boot the system from the disk and run uFlash
Type the name of the Firmware (with .bin).
The drive will now flashed.
If the flash ended you can reboot the system with new firmware.

If the drive not answer within 10 seconds the flashing was failed and the drive is dead! It can only recovered by desoldering the flash and reprogram it outside of the drive. (very Expensive tools needed).
