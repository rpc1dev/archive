I was often asked how to change the Master/Slave setting of the QSI Drives?

The official QSI Flashtools has a undocumented commandline switch to change this setting.

/Tmmaster
/Tsslave
/Tccable select
/Trreverse csel

Flashing your drive! (Works only witch real DOS-MODE)

Create a Bootdisk and unzip Flashtool and a firmware to the root directory of the bootdisk.
Boot the system from the disk
First you should make a backup of your current Firmware

A:\> QFSBW081 /OR /Fbackup.bin (Enter)

Now open the file with an HexViewer/Editor and search for "QSI01" after this you find the complete firmware version. IMHO no tool shows this. Download the patched version. Please switch off your computer and start again to flash the new firmware

A:\> QFSBW081 /Ffile.bin (Enter)

The drive will now flashed.
If the flash ended you can reboot the system with new firmware.

If you have an other firmware please make only the backup and send it to me.
