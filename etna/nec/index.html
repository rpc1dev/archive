I suggest to use the Autoreset firmwares because with this no aditional software like Region Killer is needed. After every startup the changes are set to 5 and the region to not set. (The drive takes the region of the insertet disc).

Flashing your drive! (Works only with real DOS-MODE)


Create a Bootdisk and unzip Flashtool and a firmware to the root directory of the bootdisk.
Boot the system from the bootdisk (Remove all medias from the nec)

The config switches are:
-pri (Primary) -sec (Secondary) -mas (Master) and -sla (Slave)

First you should make a backup of your current Firmware

A:\> NEC1100A (config switches) -out original.bin (Enter)

Check if the file has 620KB (some old versions are only 492KB) if this is smaller repeat the backup procedure. Now open the file with an HexViewer/Editor and search for "WRT_str" after this you find your servo version (SP5_6 = 5 or SP7 = 7)

With this info you can select the correct firmware for your drive. (I'm not sure if this is important because some users have succsessful updated an ND1000 (5) to 1.A0 (7)

Flash it with

A:\> NEC1100A (config switches) -flash newfirm.bin (Enter)

 The drive will now flashed.
If the flash ended you can reboot the system with new firmware.

If you have an other firmware please make only the backup and send it to me.

Some words about upflashing:

ND1000 to ND1100 seems to be working
ND1100 to ND1300 is partially working but only with a special hack (I had some probs with DVD-R writing but maybe the original 1300 has similar probs.)
